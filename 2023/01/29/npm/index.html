NPM命令速记

发表于2023-01-29|更新于2023-01-29|前端
|字数总计:449|阅读时长:1分钟|阅读量:|评论数:

  • npm即为node包管理工具，是node.js默认的、以JavaScript编写的软件包管理系统

  • 设置淘宝镜像

    npm config set registry https://registry.npm.taobao.org --global

    npm config set disturl https://npm.taobao.org/dist --global

  • 设置当前的地址（设置为默认地址）

    npm config set registry https://registry.npmjs.org/

  • 查看镜像和配置结果

    npm config get registry

    npm config get disturl

  • 使用nrm工具切换淘宝源

    npm nrm use taobao

  • 切换回官方源

    npx nrm use npm

  • 查看全局包安装位置

    npm root -g

  • 查看版本

    npm -v

  • 安装模块/全局安装

    npm install <module name> -g

  • 查看所有全局安装的模块

    npm list -g

  • 查看某模块的版本号

    npm list <module name>

  • 更新npm某个版本(@后指定版本)

    npm -g install npm@5.9.1

  • 在package文件的dependcies（运行时依赖）节点写入依赖（默认）

    npm install -save <modulename>

  • 在package文件的devDependcies（开发时依赖）节点写入依赖

    npm install -save-dev <modulename>

  • 生成package.json配置文件

    npm init

package文件的内容

当下载了别的包后，自己创建的包名出现最后两行：

分别是开发依赖和运行时依赖，如果拷贝给别人不带module包，那么拷贝后执行：npm install会自动下载所需的依赖包。

值得注意的是：jquery: '^2.9.0'是初始版本为2.9.0， 但是会更新后两位的版本号到最新（如果有的话），若为~，则更新最后一位版本号到最新（如果有的话）。

  • name：包名
  • version：版本号
  • description：包的描述
  • homepage：包的官网url
  • author：作者
  • contributors：其他贡献者姓名
  • dependencies：依赖包列表
  • repository：包存放地方类型，可以为git、svn
  • main：主程序入口文件默认index.js
  • keyword：关键字
